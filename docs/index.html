LangBoost

LangBoostは、ニュース記事を活用した英語学習支援アプリです。

📝 主な機能

  • ニュースURLまたはテキストを貼り付けると自動で要約
  • TOEICスコアに応じた単語を抽出し、日本語訳・例文を表示
  • 単語帳への登録、復習クイズ機能（Leitnerシステム対応）
  • NGSL単語との照合による学習効率の最適化
